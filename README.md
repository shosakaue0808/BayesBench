# BayesBench
BayesBench is a reproducible benchmark platform for sample-efficient machine learning. It studies how Bayesian Optimization and uncertainty-aware active learning choose expensive evaluations under limited budgets. The project begins with synthetic black-box optimization, then extends to hyperparameter optimization and active learning benchmarks. The goal is to compare algorithms such as Random Search, GP-Expected Improvement, GP-UCB, Thompson Sampling, uncertainty sampling, and query-by-committee using clean, reproducible experiments.
